이름

env - 프로그램을 다른 환경에서 실행한다.  

개요

env [-] [-i] [-u name] [--ignore-environment] [--unset=name] [--help] [--version] [name=값]... [명령 [인수...]]
env  

설명

이 맨페이지는 GNU 버전의 env 를 설명한다. env 는 명령행에서 주어진 환경에서 명령을 수행한다. `variable=value' 형태의 인수들은 환경 변수 variable 를 value 로 설정한다. value 는 생략할 수 있다. (`variable='). 변수 값을 생략한다는 것은 변수를 없애는 것과 다르다.

처음으로 남는 인수는 실행할 프로그램명이다; 프로그램은 PATH 환경 변수를 따라 검색한다. 그 뒤에 나오는 값들은 모두 프로그램에 대한 전달 인수로 사용된다.

환경 설정 후 아무런 명령도 없으며 결과 환경만 출력된다. 명령 부분에 `printenv'라고 적은 것과 같은 결과이다.  

옵션

--help
표준출력으로 사용법을 출력하고 정상적으로 종료한다.
-u, --unset name
만약 존재한다면 환경으로부터 name 변수를 제거한다.
-, -i, --ignore-environment
상속된 환경을 무시하고 텅빈 환경에서 시작한다.
--version
표준출력으로 버전 정보를 출력하고 정상적으로 종료한다.
