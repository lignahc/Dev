이름

true - 아무 것도 하지 않고 항상 참을 반환한다.  

개요

true
true {--help,--version}  

설명

이 맨페이지는 GNU 버전의 true (아마 다른 버전들도 모두 같지 않겠는가?). 를 다룬다. true 은 아무 일도 하지 않으면 단지 ``성공''을 의미하는 0 의 종료 상태를 반환한다. 쉘의 내장 명령인 `:' (콜론)이 같은 일을 더 빠르게 하긴 하지만 쉘 스크립트 안에서 참의 반환값이 필요한 경우 사용된다.  

옵션

GNU true 가 단 하나의 인수로 실행되면 다음 옵션이 인식된다:
--help
표준출력으로 사용법을 출력하고 정상적으로 종료한다.
--version
표준출력으로 버전정보를 출력하고 정상적으로 종료한다.
