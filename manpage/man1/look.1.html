NAME

look - 주어진 문자열로 시작하는 줄을 보여준다.  

SYNOPSIS

look [-dfa ] [-t termchar ] string [file ]  

DESCRIPTION

look 명령은 file 에서 string 으로 시작하는 줄을 찾아 보여준다. 이진파일 검색일 경우에는, file 안의 각 줄이 먼저 정열되어야한다.

file 이 지정되지 않으면, /usr/dict/words 파일을 사용한다. 이 파일은 단지 알파벳 단어만 있으며, 대소문자는 무시된다. (이것은 영문 맞춤법 검사에 유용하게 쓰인다.)

 

OPTIONS

-d
영문자만을 검색한다.
-f
대소문자를 구분하지 않는다.
-a
대체 사전인 /usr/dict/web2 파일을 사용한다.
-t
Specify a string termination character, i.e. only the characters in string up to and including the first occurrence of termchar are compared.

look 명령은 오류 없이 끝나면 0, 오류가 있으면 1보다 큰 수를 리턴하고 마친다.  

FILES

/usr/dict/words
사전
/usr/dict/web2
대체 사전

 

SEE ALSO

grep(1), sort(1)  

COMPATIBILITY

원래의 매뉴얼 페이지에는 -d 옵션이 지정되었을 때, 탭문자나 공백문자도 검색 대상이 된다고 쓰여져있다. 이 부분은 틀린 부분이었기에, 여기서 언급한다.  

HISTORY

Look 명령은 Version 7 AT&T Unix에서 나타났다.
