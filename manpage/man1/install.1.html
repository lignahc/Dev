NAME

install - 지정한 속성으로 파일을 복사한다.  

SYNOPSIS

install [options] [-s] [--strip] source dest
install [options] [-s] [--strip] source... directory
install [options] [-d,--directory] directory...
Options:
[-c] [-g group] [-m mode] [-o owner] [--group=group] [--mode=mode] [--owner=owner] [--help] [--version]  

DESCRIPTION

이 문서는 더이상 최신 정보를 담고 있지않다. 그래서, 몇몇 틀릴 경우도 있고, 부족한 경우도 있을 것이다. 완전한 매뉴얼을 원하면, Texinfo 문서를 참조하기 바란다.

이 매뉴얼 페이지는 install 명령의 GNU 버전에 대한 것이다. install 명령은 파일의 권한 속성들, 소유주, 그룹을 지정해서 복사한다. cp 명령과 비슷하나, 엄격히 말해, 이 명령은 주로, Makefile에서 사용된다. 이 명령은 필요하다면, 그 파일이 복사될 경로까지도 만든다. 물론 원본과 대상이 완전히 일치하게 install할 수는 없다. (`install dest dest' 이런 명령은 허용되지 않는다는 말)  

OPTIONS

-c
무시; 옛날 유닉스 버전과의 호환을 위해서 사용된다.
-d, --directory
만약 지정한 대상에서 경로가 사용되는데, 이 경로가 없다면, 그 경로까지 만든다. 이 경로의 모드, 소유주, 그룹은 명령행에서 지정한 것을 그대로 사용한다. 또한 그 하위에 있는 모든 경로도 같다. 이것은 이렇게 파일 속성을 그대로 종속한다는 점에서, SunOS 4.X 버전의 install 명령과 다른 점이다.
-g, --group group
지정한 group ID나 이름으로 그룹을 지정한다. (초기값은 현재 그룹)
-m, --mode mode
모드값 지정. chmod에서 사용되는 방법과 같음. 초기값은 0755
-o, --owner owner
root로 실행한다면, 소유주를 변경할 수 있다. 이 값은 ID나 이름이 올 수 있다. (초기값은 root)
-s, --strip
strip명령을 수행한다.(심블 테이블을 지움 - 파일 크기가 줄어듬)
--help
도움말을 보여주고 마친다.
--version
버전 정보를 보여주고 마친다.
