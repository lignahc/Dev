이름

basename - 화일명으로부터 디렉토리와 suffix 를 제거한다.  

개요

basename name [suffix]
basename {--help,--version}  

설명

이 맨페이지는 GNU 버전의 basename 에 대하여 설명한다. basename 명령은 name 로부터 앞부분의 디렉토리명 부분을 모두 제거한다; 만약 suffix 이 주어졌으며 name 의 끝부분과 같다면, 이것 또한 name 로부터 제거된다. basename 은 결과 화일명을 표준출력으로 출력한다.  

옵션

--help
표준출력으로 사용법을 출력하고 정상적으로 종료한다.
--version
표준출력으로 버전 정보를 출력하고 정상적으로 종료한다.
