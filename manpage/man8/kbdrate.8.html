이름

kbdrate - 글쇠 누름 속도에 대해서 다시 지정한다.  

사용법

kbdrate [ -s ] [ -r rate ] [ -d delay ]  

설명

kbdrate 풀그림은 IBM 글쇠판의 rate와 delay 값을 바꾸는데 사용된다.

아무 옵션 없이 사용하면, 초기값으로 rate는 10.9, delay는 250으로 지정된다. 이것이 IBM 표준이다.  

옵션

-s
아무 메시지를 보여주지 않고 끝낸다.
-r rate
글쇠 rate 값을 바꾼다. rate 란, 1초 동안 찍을 수 있는 글자 수를 말한다. rate 값은 2.0부터 30.0까지다. 이 값이 지정되면, 시스템에서 처리할 수 있는 가장 적당한 값으로 변환되어 그 값이 지정된다. 실질적으로 지정할 수 있는 값은 다음과 같다: 2.0, 2.1, 2.3, 2.5, 2.7, 3.0, 3.3, 3.7, 4.0, 4.3, 4.6, 5.0, 5.5, 6.0, 6.7, 7.5, 8.0, 8.6, 9.2, 10.0, 10.9, 12.0, 13.3, 15.0, 16.0, 17.1, 18.5, 20.0, 21.8, 24.0, 26.7, 30.0.
-d delay
글쇠의 delay 값을 바꾼다. delay란 글쇠를 누르고 있을 때, 이것이 연속 글쇠 누름이냐, 아니면, 단순 글쇠 누름이냐를 판단하는 밀리초(1/1000초)이다. delay 값은 250부터 1000까지다. 실질적으로 사용될 수 있는 값은 다음과 같다: 250mS, 500mS, 750mS, and 1000mS.
 

BUGS

모든 글쇠판이 모든 rate를 지원하지는 않는다.

모든 글쇠판이 똑같은 rate 값을 사용하지는 않는다.

Gateway AnyKey keyboard에서는 rate 지정이 바르게 작동되지 않는다. 만약, 누군가가 Gateway AnyKey keyboard의 글쇠 처리 방법을 알고 있다면, faith@cs.unc.edu로 편지를 부탁한다.
