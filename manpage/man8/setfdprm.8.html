이름

setfdprm - 플로피 디스크 사양을 사용자가 직접 설정한다.  

사용법

setfdprm [ -p ] 장치이름 이름
setfdprm [ -p ] device size sectors heads tracks stretch gap rate spec1 fmt_gap
setfdprm [ -c ] 장치이름
setfdprm [ -y ] 장치이름
setfdprm [ -n ] 장치이름  

설명

setfdprm 풀그림은 플로피 장치를 자동 검색할 수 있도록 디스크 사양을 지정하거나, 옛 사양을 지우고, 검사 메시지를 보여줄 것인지, 보여주지 않을 것인지 지정하는데 사용된다.

아무 옵션 없이, setfdprm 명령을 사용하면, /etc/fdprm 파일에서 이름을 찾아, 그에 해당되는 나머지 정보들을 사용해서 장치이름을 가진 플로피 디스크 사양을 지정한다. 이름은 대게 360/360 이런 식이고, 장치이름은 대게 /dev/fd0 또는 /dev/fd1 이런 식이다. 이 인자들은 매체가 바뀌기 전까지 계속 그 영향이 지속된다.

-p 장치이름 이름
이 옵션을 사용하면, 아에 /etc/fdprm 파일의 정보를 바꾸어 버린다. 이렇게 되면 다음번 플로피 디스크 자동 검색에서 여기서 지정한 값을 사용하게 된다. 물론 명령행에서 지정하는 각각의 정보가 그대로 지정된다.
-c 장치이름
플로피 디스크 장치의 자동 환경 설정 사양을 지운다.
-y 장치이름
플로피 디스크 자동 환경 설정을 지정한 포멧 검색 메시지를 사용 가능 상태로 한다.
-n 장치이름
플로피 디스크 자동 환경 설정을 지정한 포멧 검색 메시지를 사용 불가 상태로 한다.
 

벌레

관련 파일
