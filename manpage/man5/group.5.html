이름

group - user group file  

설명

/etc/group은 사용자들이 속한 그룹을 규정하는 ASCII 파일이다. 한 줄에 한명씩 있고, 각 줄마다 형식을 가지고 있다:

group_name:passwd:GID:user_list

각 부분이 뜻하는 것은 아래와 같다:

group_name
그룹의 이름.
password
(암호화 된)그룹 패스워드. 이 부분이 비어있다면, 패스워드가 필요 없는 것이다.
GID
숫자로 된 그룹ID.
user_list
그룹 구성원 모두의 사용자 이름,콤마로 구분된다.
 

파일

/etc/group  

관련 항목

login(1), newgrp(1), passwd(5)
