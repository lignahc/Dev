NAME

initscript - inittab 명령을 실행하는 스크립트.  

SYNOPSIS

/bin/sh /etc/initscript id runlevels action process  

DESCRIPTION

/etc/initscript 파일이 존재하면, init는 inittab으로 부터 명령을 실행하기 위해 이 파일을 사용한다. 이 스크립트는 모든 프로세스를 위한 ulimit나 umask 초기값같은 것을 지정하는데, 사용될 수 있다.  

EXAMPLES

아래는 initscript의 한 예제이다. 이 파일은 자신의 시스템에 /etc/initscript.sample 파일로 있을 것이다. 


#
# initscript   Executed by init(8) for every program it
#              wants to spawn like this:
#
# /bin/sh /etc/initscript <id> <level> <action> <process>
#

# Set umask to safe level, and enable core dumps.
umask 022
ulimit -c 2097151
PATH=/bin:/sbin:/usr/bin:/usr/sbin
export PATH

# Execute the program.
eval exec "$4"
