이름

isatty - 이 기술자가 터미널을 지시하는지 검사한다.  

사용법

#include <unistd.h>

int isatty ( int desc );
 

설명

만일 desc가 터미널에 연결된 열린 기술자라면 1을 그렇지 않으면 0을 반환한다.  

호환

SVID, AT&T, X/OPEN, BSD 4.3  

관련 항목

fstat(2), ttyname(3)
