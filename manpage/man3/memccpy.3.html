NAME

memccpy - 메모리 영역 복사하기.  

SYNOPSIS

#include <string.h>

void *memccpy(void *dest, const void *src, int c, size_t n);
 

DESCRIPTION

memccpy() 함수는 메모리 영역 src에서 메모리 영역 dest로 단지 n 바이트만을 복사하며, 문자 c가 발견되었을때 멈춘다.

 

RETURN VALUE

memccpy() 함수는 c 다음에 dest중 다음 문자에 대한 포인터를 반환한다. 만일 c가 src의 처음 n개의 문자들중에 발견되지 않는다면 NULL을 반환한다.

 

CONFORMING TO

SVID 3, BSD 4.3

bcopy(3), memcpy(3), memmove(3), strcpy(3), strncpy(3)
