NAME

memcmp - 메모리 영역 비교하기  

SYNOPSIS

#include <string.h>

int memcmp(const void *s1, const void *s2, size_t n);
 

DESCRIPTION

memcmp() 함수는 메모리 영역 s1과 s2의 처음 n 바이트를 비교한다. 이 함수는 s1이 s2보다 작거나 같거나 크다면, 각각 0보다 작은 정수, 0, 0보다 큰 정수를 반환한다.  

RETURN VALUE

memcmp() 함수는 s1의 처음 n 바이트가 s2의 처음 n 바이트보다 작거나 같거나 크다면, 각각 0보다 작은 정수, 0, 0보다 큰 정수를 반환한다.  

CONFORMING TO

SVID 3, BSD 4.3, ISO 9899

bcmp(3), strcasecmp(3), strcmp(3), strcoll(3), strncmp(3), strncasecmp(3)
