이름

cbrt - 세제곱 루트 함수  

사용법

#include <math.h>

double cbrt ( double x );
 

설명

cbrt() 함수는 x의 세제곱 루트 값을 반환한다. 이 함수는 실패할수 없다; 모든 표현 가능한 실수 값은 표현할수 있는 실 세제곱 루트값을 가진다.  

호환

cbrt 는 GNU 확장이다.  

관련 항목

sqrt(3), pow(3)
