이름

labs - long 형 정수형의 절대 값을 계산한다.  

사용법

#include <stdlib.h>

long int labs(long int j);
 

설명

labs() 함수는 long형 정수 인자 j의 절대 값을 계산한다.  

반환값

long 형 정수 인자의 절대값을 반환한다.  

호환

SVID 3, BSD 4.3, ISO 9899  

주의

대부분의 음의 정수의 절대값을 가질려는 시도는 정의되지 않는다.  

관련 항목

abs(3), ceil(3), floor(3), fabs(3), rint(3)
