이름

ttyname - 터미널 이름을 반환한다.  

사용법

#include <unistd.h>

char *ttyname ( int desc );
 

설명

파일 기술자 desc를 열은 터미널 장치의 경로이름에 대한 포인터를 반환하거나 에러(예를 들어, 만일 desc 가 터미널에 연결되어 있지 않다면) 라면 NULL을 반환한다.  

호환

POSIX.1  

관련 항목

fstat(2), isatty(3)
