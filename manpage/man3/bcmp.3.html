이름

bcmp - 바이트 문자열 비교  

사용법

#include <string.h>

int bcmp(const void *s1, const void *s2, size_t n);
 

설명

bcmp() 함수는 s1 and s2 스트링의 처음 n바이트를 비교한다. 두 스트링이 같으면, bcmp() 는 0을 반환하고, 그렇지 않으면 non-zero result를 반환한다. n이 0이면, 두게의 스트링은 같은 것이다.  

반환값

bcmp() 함수는 두 스트링이 같으면 0을 반환하고, 그렇지 않으면 non-zero result를 반환한다..  

호환

4.3BSD. This function is deprecated -- use memcmp in new programs.  

관련 항목
