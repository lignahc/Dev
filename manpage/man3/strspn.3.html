이름

strspn, strcspn - 문자열을 문자들의 집합에서 찾는다.  

사용법

#include <string.h>

size_t strspn(const char *s, const char *accept);

size_t strcspn(const char *s, const char *reject);
 

설명

strspn() 함수는 전체적으로 accept에 있는 문자들로 이루어져 있는 s의 초기 세그먼트의 길이를 계산한다.

strcspn() 함수는 전체적으로 reject에 없는 문자들로 이루어져 있는 s의 초기 세그먼트의 길이를 계산한다.  

반환값

strspn() 함수는 accept에서 문자들로만 이루어져 있는 s의 초기 세그먼트에 있는 문자들의 수를 반환한다.

strcspn() 함수는 문자열 reject에는 없는 s의 초기 세그먼트에 있는 문자들의 수를 반환한다.  

호환

SVID 3, POSIX, BSD 4.3, ISO 9899
