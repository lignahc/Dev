NAME

memchr - 문자를 위해 메모리를 스캔한다.  

SYNOPSIS

#include <string.h>

void *memchr(const void *s, int c, size_t n);
 

DESCRIPTION

memchr() 함수는 문자 c를 위해 s가 가리키는 메모리 영역의 처음 n 바이트를 스캔한다. c(unsigned character로 해석)와 일치하는 첫 바이트에서 작동을 멈춘다.  

RETURN VALUE

memchr() 함수는 일치하는 바이트에 대한 포인터를 반환하거나, 만일 문자가 주어진 메모리 영역에 있지 않다면 NULL을 반환한다.  

CONFORMING TO

SVID 3, BSD 4.3, ISO 9899
