이름

bzero - 0 을 바이트 스트링에 쓴다.  

사용법

#include <string.h>

void bzero(void *s, size_t n);
 

설명

bzero() 함수는 바이트 스트링 s의 처음 n 바이트를 0으로 설정한다.  

반환값

bzero() 함수는 어떤 값도 반환하지 않는다.  

호환

4.3BSD. 이 함수는 구식이 되었다.--새 프로그램에서는 memset을 사용해라.  

관련 항목

memset(3), swab(3)
