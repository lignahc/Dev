이름

strcasecmp, strncasecmp - 대 소문자를 구분하지 않고 두개의 문자열을 비교  

사용법

#include <string.h>

int strcasecmp(const char *s1, const char *s2);

int strncasecmp(const char *s1, const char *s2, size_t n);
 

설명

strcasecmp() 함수는 대 소문자를 구분하지 않고, 두 문자열 s1과 s2를 비교한다. 이 함수는 만일 s1가 s2보다 작거나 일치하거나 큰 경우에 각각 0 보다 작은 정수, 0, 0보다 큰 정수를 반환한다.

strncasecmp() 함수는 s1의 문자들 중 처음 n만을 비교한다는 것을 제외하고 비슷하다.  

반환값

strcasecmp()와 strncasecmp() 함수는 만일 s1(또는 처음 n 바이트)가 s2보다 작거나 일치하거나 큰 경우에 각각 0 보다 작은 정수, 0, 0보다 큰 정수를 반환한다.  

호환

BSD 4.4  

관련 항목

bcmp(3), memcmp(3), strcmp(3), strcoll(3), strncmp(3)
