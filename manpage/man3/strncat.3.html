이름

strcat, strncat - 두 문자열을 연결시킨다.  

사용법

#include <string.h>

char *strcat(char *dest, const char *src);

char *strncat(char *dest, const char *src, size_t n);
 

설명

strcat() 함수는 dest의 마지막 `\0' 문자를 겹쳐서 src 문자열을 dest 문자열에 추가시킨다. 그리고 마지막에 `\0' 문자를 추가시킨다. 문자열은 중첩되지 않으며 dest 문자열은 결과를 위해 충분한 공간을 가지고 있어야 한다.

strncat() 함수는 src의 처음 n문자들만 dest에 추가시키는것을 제외하고는 비슷하다.  

반환값

strcat()와 strncat() 함수는 결과 스트링 dest에 대한 포인터를 반환한다.  

호환

SVID 3, POSIX, BSD 4.3, ISO 9899  

관련 항목
