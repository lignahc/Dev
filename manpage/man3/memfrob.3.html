NAME

memfrob - 메모리 영역을 암호화 한다.  

SYNOPSIS

#include <string.h>

void *memfrob(void *s, size_t n);
 

DESCRIPTION

memfrob() 함수는 메모리 영역 s의 처음 n를 숫자 42로 각 문자를 exclusive-ORing하여 암호화 한다. 결과는 암호화된 영역에서 memfrob()를 사용하여 원래대로 할수 있다.

이 함수는 XOR 상수를 고정시켰기 때문에 적당한 암호화 루틴이 아니며 문자들을 숨기는데만 알맞다는 점을 주의해라.  

RETURN VALUE

memfrob() 함수는 암호화된 영역에 대한 포인터를 반환한다.  

CONFORMING TO

memfrob() 함수는 Linux C 라이브러리와 GNU C 라이브러리에서 유일하다. GNU C Library.
