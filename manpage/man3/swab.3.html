이름

swab - 근접 바이트들을 교환한다.  

사용법

#include <string.h>

void swab(const void *from, void *to, size_t n);
 

설명

swab() 함수는 from이 가리키는 배열에서 to가 가리키는 배열로 n 바이트를 복사하고, 근접해 있는 짝수와 홀수 바이트들을 교환한다. 이 함수는 다른 하위/상위 바이트 순서를 가지고 있는 머신들 사이에서 데이터를 교환하는데 사용된다.  

반환값

swab()함수는 어떤 값도 반환하지 않는다.  

호환

SVID 3, BSD 4.3  

관련 항목

bstring(3)
