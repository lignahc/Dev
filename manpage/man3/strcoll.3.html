이름

strcoll - 현재 로케일을 사용하여 두 문자열을 비교한다.  

사용법

#include <string.h>

int strcoll(const char *s1, const char *s2);
 

설명

strcoll() 함수는 두 문자열 s1과 s2를 비교한다. 이 함수는 만일 s1이 s2보다 작거나 같거나 클때, 각각 0 보다 작은 정수나 0, 0 보다 큰 정수를 반환한다. 비교는 문자열에 기반하며 프로그램의 현재 로케일 카테고리 LC_COLLATE 에 맞게 해석된다. (setlocale(3) 참조).  

반환값

strcoll() 함수는 두 문자열이 현재 로케일에 맞게 해석될때, 만일 s1이 s2보다 작거나 같거나 클때, 각각 0 보다 작은 정수나 0, 0 보다 큰 정수를 반환한다.  

호환

SVID 3, BSD 4.3, ISO 9899  

주의

"POSIX" 나 "C" 로케일에서, strcoll()는 strcmp()와 동일하다.  

관련 항목

bcmp(3), memcmp(3), strcasecmp(3), strcmp(3), strxfrm(3), setlocale(3)
