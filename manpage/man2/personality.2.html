이름

personality - 프로세스 실행 도메인 설정.  

사용법

int personality(unsigned long persona);  

설명

리눅스는 프로세스별로 각기 다른 실행 도메인을 제공한다. 다른 것들 가운데, 실행 도메인은 리눅스가 시그널 액션에 어떻게 시그널 번호를 매핑하는지 알려준다. 실행 도메인 시스템은 리눅스가 다른 유닉스 호환 시스템하의 바이너리 컴파일의 제 한된 지원을 한다. personality 는 현재 프로세스의 새로운 실행 도메인인 persona 를 참조해 실행 도메인을 만들것이다.  

반환값

성공시, persona 는 새로운 실행 도메인을 만들고 이전 persona 를 반환한다. 실패시, -1을 반환하고 errno 가 적절히 설정된다.  

에러

EINVAL
persona 는 유효한 실행 도메인이 아니다.

 

파일

/usr/include/linux/personality.h  

호환

personality 는 리눅스에서만 사용되며 다른 프로그램에서는 사용할 수 없다.
