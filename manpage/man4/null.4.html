이름

null, zero - 데이터 소멸점(sink)  

설명

null나 zero 특수 파일에 쓴 데이터는 버려진다.

null 특수 파일에서 읽으면 항상 EOF를 리턴하는 반면, zero에서 읽으면 항상 \0 문자를 리턴한다.

null와 zero는 다음과 같이 만들어진다:

mknod -m 666 /dev/null c 1 3
mknod -m 666 /dev/zero c 1 5
chown root:mem /dev/null /dev/zero

 

주의

이 장치가 모든 사용자에게 쓰기와 읽기 가능하지 않으면, 많은 프로그램이 이상한 동작을 할 것이다.  

파일

/dev/null
/dev/zero  

관련 항목

mknod(1), chown(1)
